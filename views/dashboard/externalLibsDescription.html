Using the Image Resizer with Responsive libs

You can set bounds for each image with the constrainHeight and/or constrainWidth commands and using the resized image for different device size. How those bounds are interpreted is determined by the constrainMode. constrainMode can have following values:

  • fit – fit source image to new size;
  • fill – fill source image to new size (by default);
  • original – resized image preserves the aspect ratio;
